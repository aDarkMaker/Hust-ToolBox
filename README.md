## 📖 项目简介

Hust-ToolBox 是一个专为华中科技大学（HUST）学生设计的综合性工具包，旨在简化日常学习和生活中的各种操作，提高效率。

## ✨ 功能特性

### 🌐 网站自动化工具

- **自动评教系统** - 快速完成课程评价
- **Hub 系统自动登录** - 免去重复登录烦恼
- **MOOC 课程助手** - 自动刷课、答题功能

### 💻 校园生活助手

- **校园跑步脚本** - 自动完成跑步任务
- **微助教签到** - 智能课堂签到

### 🔧 实用小工具

- **防粘贴锁定** - 突破网页复制限制
