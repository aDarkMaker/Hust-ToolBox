# HUST 验证码自动识别

一个用于华中科技大学统一身份认证系统的验证码自动识别脚本。

## 功能

- 自动识别登录页面的验证码图片
- 实时填充验证码输入框
- 支持动态 GIF 验证码识别

## 使用方法

1. 安装脚本管理器(如 Violentmonkey)到浏览器
2. 安装此脚本
3. 访问 Hub 登录系统
4. 脚本会自动识别并填充验证码
