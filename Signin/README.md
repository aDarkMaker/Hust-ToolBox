## 使用说明

No matter which method you choose, you need to get OpenId first.

### How to get OpenId ：

（以下方法均发生在电脑端）

1. 打开浏览器进行一次签到活动;
2. 手快直接趁未加载完成转发连接，获取 OpenId;
3. 转发连接至微信桌面版，获取连接（成功率最高）;
4. 企业微信法据说可行，没有测试，方法同 Method 3;
